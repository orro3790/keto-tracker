updateDiet(userId, macros)

Updates the user's diet. Also updates goal snapshots and precision for any entries >= today's date.

Parameters:
Name Type Description
userId string

The current user's Id.
macros object

The user's complete diet settings. All keys must be present, including the user's water goals. Any optional macros that are not being tracked must be set as null values.
Source:

macros = {

k: null,
e: 2000,

p: 100,
w: null
}

updateGoalsAndPrecision(entry, goals)

Updates the goal snapshot and precision values for a single food diary entry. Diet and water snapshots are updated if the entry date is not in the past. Precision is always recalculated.
